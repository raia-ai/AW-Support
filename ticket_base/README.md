## How to Use This Database

1. **Search by keyword** — Use the GitBook search bar to find tickets matching your issue.
2. **Browse by category** — Navigate to a category page below to see all tickets in that area.
3. **Search by ticket ID** — If you have a specific ticket number (e.g., `AW-235279`), search for it directly.

## Schichten

### Arbeitsfreie Tage anlegen (KAP 01-001)

**Ziel der Handlung**
Sie definieren Tage, an denen nicht gearbeitet wird.

**Voraussetzungen**
- Schicht muss angelegt sein.

**Zusatzinfo**
Definieren Sie hier arbeitsfreie Tage (z.B. Feiertage), für die keine Schichten angelegt werden sollen. Um neue Einträge anzulegen, klicken Sie bitte auf "Neu". Bei Nutzung des ERP-WebService können Sie aber auch zur Schichtenerzeugung den Kalender aus dem ERP-System nehmen.

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Schichten**.
2. Markieren Sie im Feld **Schichteditor** die Option **Kalender**.
3. Klicken Sie auf **[Neu]**. Im Bereich **Kalender** wird eine neue Zeile eingefügt.
4. Bearbeiten Sie den arbeitsfreien Tag, indem Sie die Felder markieren und **Datum** und **Kommentar** ändern.
5. Klicken Sie auf **[Übernehmen]**, um die Daten zu speichern.

### Arbeitsfreie Tage bearbeiten (KAP 01-002)

**Ziel der Handlung**
Sie nehmen Korrekturen für Tage vor, an denen nicht gearbeitet wird.

**Voraussetzungen**
- Arbeitsfreie Tage müssen definiert sein.

**Zusatzinfo**
Definieren Sie hier arbeitsfreie Tage (z. B. Feiertage), für die keine Schichten angelegt werden sollen. Bei Nutzung des ERP-WebService können Sie aber auch zur Schichtenerzeugung den Kalender aus dem ERP-System nehmen.

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Schichten**.
2. Markieren Sie im Bereich **Schichteditor** die Option **Kalender**.
3. Markieren Sie im Feld **Kalender** ein Feld, das Sie bearbeiten möchten.
4. Ändern Sie die Werte.
5. Klicken Sie auf **[Übernehmen]**, um die Daten zu speichern.

### Arbeitsfreie Tage löschen (KAP 01-003)

**Ziel der Handlung**
Sie löschen Tage, an denen nicht gearbeitet wird.

**Voraussetzungen**
- Arbeitsfreie Tage müssen definiert sein.

**Zusatzinfo**
Definieren Sie hier arbeitsfreie Tage (z. B. Feiertage), für die keine Schichten angelegt werden sollen. Bei Nutzung des ERP-WebService können Sie aber auch zur Schichtenerzeugung den Kalender aus dem ERP-System nehmen.

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Schichten**.
2. Wählen Sie im Bereich **Kalender** eine Zeile, indem Sie sie in der ersten Spalte markieren.
3. Klicken Sie auf **[Löschen]**.
4. Klicken Sie auf **[Übernehmen]**, um die Daten zu speichern.

## Schichtpläne

### Schichtplan erstellen (KAP 02-001)

**Ziel der Handlung**
Sie erstellen einen Schichtplan.

**Voraussetzungen**
- Schicht muss angelegt sein.

**Zusatzinfo**
Ein Schichtplan kann erst gespeichert werden, wenn ihm eine Schichtregel und eine Schichtgruppe mit Maschinen zugeordnet wurden!

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Schichten**.
2. Markieren Sie im Bereich **Schichteditor** nicht die Option **Kalender**.
3. Klicken Sie auf die Auswahl-Liste **[Neu]**.
4. Wählen Sie **Neuer Schichtplan**. Im Bereich **Schichteditor** wird ein neuer Schichtplan mit der Bezeichnung "Neu" angelegt. Alternativ öffnen Sie mit einem Rechtsklick auf den Kalender das Kontext-Menü und wählen den Eintrag **Neuer Schichtplan**.
5. Geben Sie dem neuen Schichtplan im Feld **Schichtplan** im Feld **Name** eine Bezeichnung, z. B. "Zwei-Schicht normal".
6. Klicken Sie auf **[Übernehmen]**, um die Daten zu speichern.
7. Klicken Sie auf **[Erzeugen]**, damit die Daten des Schichtplans erzeugt werden.

### Schichtplan bearbeiten (KAP 02-002)

**Ziel der Handlung**
Sie nehmen Änderungen an einem Schichtplan vor.

**Voraussetzungen**
- Schichtplan muss angelegt sein.

**Zusatzinfo**
Einen Schichtplan zu bearbeiten, ist ein typischer Anwendungsfall, um bestehende Schichtpläne zu verlängern.

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Schichten**.
2. Markieren Sie im Bereich **Schichteditor** den Schichtplan, den Sie bearbeiten möchten.
3. Bearbeiten Sie die Werte des Schichtplans im Bereich **Schichtplan**.
4. Klicken Sie auf **[Übernehmen]**, um die Daten zu speichern.
5. Klicken Sie auf **[Erzeugen]**, damit die Daten des Schichtplans erzeugt werden. Die Schaltfläche **[Erzeugen]** wird erst aktiv, wenn eine Schichtregel und eine Schichtgruppe zugeordnet wurde.

### Schichtplan löschen (KAP 02-003)

**Ziel der Handlung**
Sie löschen einen bestehenden Schichtplan.

**Voraussetzungen**
- Schichtplan muss angelegt sein.

**Zusatzinfo**
Wenn Sie einen Schichtplan löschen, werden die zugehörige Schichtregel und die zugehörige Schichtgruppe mit gelöscht!

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Schichten**.
2. Markieren Sie den Schichtplan, den Sie löschen wollen.
3. Klicken Sie auf **[Löschen]**.
4. Klicken Sie auf **[Übernehmen]**, um die Daten zu speichern.

## Übergangszeiten

### Übergangszeit anlegen (KAP 05-001)

**Ziel der Handlung**
Sie legen eine Übergangszeit an.

**Voraussetzungen**
- Um Übergangszeiten anzulegen, müssen die entsprechenden logischen Maschinen in der Maschinenzuordnung definiert sein.

**Zusatzinfo**
Hier können Sie Übergangszeiten zwischen den Arbeitsstationen definieren. Wenn Sie Übergangszeiten in Stunden definieren (Dauer) verwenden Sie 4 oder 8 Stunden. Wenn Sie Übergangszeiten in Schichten definieren, verwenden Sie ganze Schichten (1, 2, 3, ...).

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Übergangszeiten**.
2. Klicken Sie auf **[Neu]**. In der Liste **Maschinen** wird oben eine neue Zeile `-1 / -1` angelegt.
3. Wählen Sie aus, von welcher logischen Maschine zu welcher logischen Maschine Sie den Übergang definieren. Nutzen Sie auch die Möglichkeiten **Alle Maschinen** auf Maschine X und Maschine X auf **alle Maschinen**. Verwenden Sie je Maschine entweder Stunden oder Schichten, nicht beides.
4. Tragen Sie in den Spalten **Maximale Dauer** oder **Maximale Schichten** einen zeitlichen Wert ein, mit dem Sie dem System erlauben die Übergangszeit maximal zu dehnen. Verwenden Sie auch hier entweder Stunden oder Schichten.
5. Klicken Sie auf **[OK]**, um die Daten zu speichern. Damit haben Sie eine neue Übergangszeit angelegt.

### Übergangszeit bearbeiten (KAP 05-002)

**Ziel der Handlung**
Sie nehmen Änderungen an einer Übergangszeit vor.

**Voraussetzungen**
- Übergangszeit muss angelegt sein.

**Zusatzinfo**
(Keine Zusatzinfo angegeben)

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Übergangszeiten**.
2. Bearbeiten Sie beliebig viele Felder in der Liste **Maschinen**. Sie aktivieren die Felder mit einem Doppelklick.
3. Klicken Sie auf **[OK]**, um die Daten zu speichern.

### Übergangszeit löschen (KAP 05-003)

**Ziel der Handlung**
Sie löschen eine bestehende Übergangszeit.

**Voraussetzungen**
- Übergangszeit muss angelegt sein.

**Zusatzinfo**
(Keine Zusatzinfo angegeben)

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Übergangszeiten**.
2. Markieren Sie in der ersten Spalte die Übergangszeit, die Sie löschen möchten. Die gesamte Zeile der Übergangszeit wird markiert.
3. Klicken Sie auf **[Löschen]**. Die Übergangszeit wird aus der Liste gelöscht.
4. **Falls Sie eine Übergangszeit versehentlich aus der Liste gelöscht haben:** Klicken Sie auf **[Ende]**. Der Dialog **Übergangszeiten** wird geschlossen und der Dialog **Änderungen speichern** wird geöffnet.
5. Klicken Sie auf **[Nein]**.
6. Klicken Sie auf **[OK]**, um den Dialog zu schließen und die Änderungen, also das Löschen der Übergangszeit, zu übernehmen.

## Vorgabezeiten

### Vorgabezeitformel anlegen (KAP 06-001)

**Ziel der Handlung**
Sie legen eine Vorgabezeitformel an.

**Voraussetzungen**
(Keine Voraussetzungen angegeben)

**Zusatzinfo**
(Keine Zusatzinfo angegeben)

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Vorgabezeiten**. Der Dialog **Vorgabezeiten** wird geöffnet.
2. Markieren Sie die logische Maschine, für die Sie eine Vorgabezeitformel erstellen möchten.
3. Klicken Sie auf **[Bearbeiten]**. Der Dialog **Vorgabezeitformel** wird geöffnet. Wenn für die logische Maschine bereits in der Maschinenzuordnung eine Formel zur Bearbeitungsdauer hinterlegt wurde, wird hier eine Warnmeldung angezeigt. Wenn Sie den Vorgang fortsetzen, wird die in der Maschinenzuordnung hinterlegte Formel überschrieben.
4. Geben Sie die **Basiszeit** in Sekunden ein.
5. Klicken Sie auf **[Neu]**, um der Vorgabezeitformel einen Faktor, einen Vektor oder ein **Festes Element** hinzuzufügen. Der Dialog **Elemente hinzufügen** wird geöffnet.
6. Wählen Sie einen Faktor, Vektor oder ein Festes Element aus der Liste. Das ausgewählte Element steht nun im Dialog **Vorgabezeitformel** in der Liste.
7. Wiederholen Sie die Schritte 5 und 6, um der Vorgabezeitformel weitere Elemente hinzuzufügen.
8. Wählen Sie mit den Checkboxen, wie die Elemente in die Vorgabezeitformel einfließen.
9. Klicken Sie auf **[OK]**, um den Dialog **Vorgabezeitformel** zu schließen und die Daten zu übernehmen. Damit haben Sie eine Vorgabezeitformel mit den benötigten Faktoren zusammengestellt.

### Eingabehilfe für Vektoren (KAP 06-002)

**Ziel der Handlung**
Sie nehmen Änderungen an Vektoren vor.

**Voraussetzungen**
(Keine Voraussetzungen angegeben)

**Zusatzinfo**
(Keine Zusatzinfo angegeben)

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Vorgabezeiten > Vorgabezeitformel**.
2. Klicken Sie auf **[...]** des Faktors, den Sie bearbeiten möchten. Der Dialog **Faktor/Vektor (Name)**, z. B. **Vektor 'Dicke -> Faktor'** wird geöffnet.
3. Klicken Sie auf **[Eingabehilfe]**. Der Dialog **Eingabehilfe für Vektoren** wird geöffnet. Zunächst geben Sie die Werte für die Stufung und dann die Werte für den Faktor ein. Aus diesen Angaben wird die Tabelle automatisch erstellt.
4. Geben Sie den **Startwert** ein, z. B. **4.00**. Dieser Wert legt fest, ab welcher Scheibendicke der Faktor gelten soll.
5. Geben Sie den **Endwert** ein, z. B. **20.00**. Dieser Wert legt fest, bis zu welcher Scheibendicke der Faktor gelten soll.
6. Geben Sie die **Schrittgröße** ein, z. B. **2.00**. Dieser Wert legt fest, in welchen Schritten die Stufung zwischen dem Startwert und den Endwert angelegt wird.
7. Geben Sie die den Startwert und die Schrittgröße für den Faktor ein, z. B **0,1**. Damit haben Sie alle notwendigen Werte eingetragen.
8. Klicken Sie auf **[OK]**, um die Daten zu speichern und den Dialog **Eingabehilfe für Vektoren** zu schließen.
9. Der Dialog **'Vektor-> Dicke'** wird geöffnet. In der Tabelle werden die Stufen mit den zugehörigen Faktoren angezeigt. Prüfen Sie, ob die Daten korrekt sind.
10. Klicken Sie auf **[OK]**, um die Daten zu speichern und den Dialog **Vektor 'Dicke -> Faktor'** zu schließen.
11. Definieren Sie im Dialog **Vorgabezeitformel** mit den Checkboxen, wie sich der Faktor auf die Vorgabezeitformel auswirken soll.

7. Klicken Sie auf **[OK]**, um die Daten zu speichern und den Dialog **Lastverteilung** zu schließen.

### Lastverteilung löschen (KAP 11-003)

**Ziel der Handlung**
Sie löschen eine Lastverteilung.

**Voraussetzungen**
- Lastverteilung muss stattgefunden haben.

**Zusatzinfo**
(Keine Zusatzinfo angegeben)

**Workflow**
1. Wählen Sie **Stammdaten > Kapazitätsplanung > Lastverteilung**.
2. Wählen Sie im Feld **Physikalische Maschinen** eine Maschine, deren Kapazität Sie löschen möchten, z. B. **910 - CNC Center**. Die zugehörigen logischen Maschinen werden in der Liste angezeigt.
3. Aktivieren Sie die Checkbox **Prüfung aktivieren**. Die logischen Maschinen können jetzt bearbeitet werden.
4. Klicken Sie doppelt in die rechte Spalte.
5. Geben Sie **000%** ein.
6. Drücken Sie die **<Eingabe>-Taste**. In dem Feld steht jetzt **unbegrenzt**, damit ist die Lastverteilung für diese logische Maschine aufgehoben.
7. Wiederholen Sie die Schritte 4 bis 6, um die restlichen Kapazitäten auf den Wert **000%** zu setzen.
8. Klicken Sie auf **[OK]**, um die Daten zu speichern und den Dialog **Lastverteilung** zu schließen. Damit haben Sie die Lastverteilung für die Maschine gelöscht.

# A+W Clarity Internal Support

This GitBook is the dedicated knowledge base for the **A+W Clarity** product ecosystem. It is intended to serve as the primary source of truth for the Clarity-specific AI support agent, enabling precise, product-focused recall without confusion from unrelated product lines.

---

## What is A+W Clarity?

**A+W Clarity** is the modern, integrated software platform for the **flat glass manufacturing** industry. It encompasses the full process chain from order entry and ERP management through production optimization, machine control, logistics, and business intelligence. The Clarity ecosystem includes the following core products and modules:

| Product | Role |
|---|---|
| **A+W Enterprise** | Core ERP for large glass manufacturers (groups and corporations) |
| **A+W Business** | ERP for small and medium-sized glass fabricators |
| **A+W Business Pro** | Combined ERP and manual production organization |
| **A+W Production** | Manufacturing execution and production optimization |
| **A+W Smart Factory** | Fully automated production control and Industry 4.0 |
| **A+W Clarity Experience** | Modern UI layer for Dispatch, Sales, and Stock Forecast |
| **A+W Smart Companion** | Mobile apps for Stock, Dispatch, and Production floor |
| **A+W Analytics** | Business intelligence powered by Qlik Sense |
| **A+W iShape** | CAD design tool for windows and shapes |
| **A+W Infrastructure / SOA** | Foundation services connecting all modules |
| **A+W iCut** | Small-scale production cutting solution |
| **A+W CapaView** | Capacity planning visualization |
| **A+W Dashboard** | Real-time KPI monitoring |

## How to Use This Knowledge Base

This GitBook is synced to the Clarity AI support agent. When a support question is received, the agent searches this knowledge base to find the most relevant documentation and provides accurate, cited answers.

**Key principles for the agent:**

1. Always cite the source document when providing an answer.
2. Confirm that the answer applies to the user's specific A+W product and version.
3. Provide step-by-step guidance where installation or configuration procedures are involved.
4. For issues not covered here, refer the user to the main A+W support portal or the master knowledge base.
5. Do not confuse Clarity ecosystem products with A+W Cantor (fenestration ERP) — those are handled by a separate agent and knowledge base.

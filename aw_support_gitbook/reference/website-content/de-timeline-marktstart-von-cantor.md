# Marktstart von CANTOR - A+W Software GmbH

by | Jan 11, 2024

![1994_cantor](https://www.a-w.com/wp-content/uploads/2024/01/1994_cantor.webp)

Das erste MS Windows-basierte Fensterbau-Softwarepaket mit einer grafischen Benutzeroberfläche (GUI) wird von A+W vorgestellt.  
6 Monate nach der Markteinführung liegen schon Verträge zur Ausstattung von zwölf großen Fensterwerken und 800 Händlern mit der CANTOR-Software vor.

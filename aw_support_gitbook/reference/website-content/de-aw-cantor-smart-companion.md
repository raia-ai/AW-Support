# A+W Cantor Smart Companion - A+W Software GmbH

## Die mobile App für Produktion und Lager

Die **A+W Smart Companion App** für A+W Cantor bietet eine smarte Lösung für Produktion und Lager, die Ihre Abläufe optimiert und gleichzeitig die Effizienz steigert.

![aw cantor smart companion](https://www.a-w.com/wp-content/uploads/2024/06/aw_cantor_smart_companion.svg "add_on_cantor_smart_companion")

Im Bereich Produktion kann die App im Versandlager genutzt werden, um Fertigelemente und Beipack auf Gestelle zu buchen und dieses einem Versandbereich zuzuordnen. Wobei mehrere Elemente gleichzeitig, im sog. Continuous Mode, gescannt werden können (Kameramodus). Als mobiles Info-Terminal (Lesemodus) unterstützt die App, indem z.B. der Produktionsstatus von einzelnen Auftragspositionen oder Objekte, die sich auf einem Gestell befinden, angezeigt werden. So ist jederzeit ersichtlich, in welchem Fertigungsschritt sich die Elemente befinden. An den zuvor festgelegten Registrierungspunkten kann Material durch Scannen der Barcodes direkt angefordert werden.

Eine weitere wichtige Funktion sind die dynamischen Formulare. Innerhalb eines Formulars können Informationen von dem Anwender abgefragt werden. Dabei ist es möglich, dass die Eingabe durch Auswahllisten, Checkboxen, Schiebebuttons etc. unterstützt wird. Diese können vom Nutzer individuell gestaltet werden. Reklamationen können in der App mit einem Foto erfasst werden und direkt an A+W Cantor CIM übermittelt werden. Somit ist keine doppelte Erfassung notwendig.

Der Bereich Lager deckt Wareneingangs- und Materialumbuchungen sowie Inventurbuchungen ab, so dass diese Prozesse vollständig über die mobile App bearbeitet werden können. So kann z. B. eine Übersicht über alle gezählten Artikel angezeigt werden. Die Daten stehen direkt in A+W Cantor zur Verfügung.

### Ihre Vorteile

Sie benötigen keinen zusätzlichen Lernaufwand, da Sie ein bekanntes, täglich genutztes Gerät verwenden.

Sie sparen Zeit dank unserer benutzerfreundlichen App und durch das schnellere Scannen mehrerer Barcodes im Continuous Mode.

Sie haben jederzeit einen Informationsvorsprung, da Sie mobil und sofort sehen können, in welchem Status sich ein Element befindet.

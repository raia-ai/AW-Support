# Setting up a calendar for the new year in A+W Business

This guide explains how to set up a new yearly calendar in A+W Business (AWB).

---
## Locating and Creating a New Calendar

The calendars in A+W Business (AWB) are set up in the master data.

1.  Navigate to the calendar settings located under `[Master Data > General > Calendar]`.

    *   **Menu Path:**
        *   Master Data
            *   General
                *   Languages
                *   Months
                *   Days
                *   Country Code
                *   **Calendar**
                *   Basic Number Ranges
            *   Products
            *   Prices

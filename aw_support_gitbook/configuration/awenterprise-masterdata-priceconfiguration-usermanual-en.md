The first part is a software reference manual covering several key areas. It begins with 'Reference Configuration,' explaining how to manage field takeovers during document entry. It then moves to 'Commission,' detailing how commission is calculated based on various parameters like product groups, customers, and time spans. 'Text Management' follows, describing how to define variable and constant text passages for use in documents and production papers. The reference section concludes with 'Costs,' which explains the system for organizing work by cost units, centers, and types. The second major part of the document is a detailed tutorial on 'Prices and Conditions.' This tutorial is aimed at users managing price calculation master data. It starts with basic concepts, a glossary of terms, and an overview of how prices and conditions interact within an order. It provides in-depth instructions on 'Price Definitions,' including the creation and use of price keys, Price List Codes (PLCD), and the logic behind their application (PLCD logics). The tutorial thoroughly covers 'Pricing Components' like rounding methods, calculation bases, and price types. A large section on 'Price Master Data' provides step-by-step workflows for creating price matrices for IG lites, price vectors for articles and processings, exchange prices, muntin prices, and surcharges.

long_description: "This document is a comprehensive technical reference for the 'Stammdaten > Firma > Firmendaten' (Master Data > Company > Company Data) section of the A+W Business ERP software. It provides step-by-step instructions and field-level explanations for configuring core company information. The manual is structured around the different tabs (Register) within the Firmendaten interface. It starts with the 'Mandant' register, detailing how to set up company clients, including address, communication data, and email server settings. The 'Steuer' (Tax) register section explains the configuration of tax office details and currency settings, including handling multiple currencies and integration with services like Avalara for tax calculation. The 'FiBu' (Financial Accounting) section describes how to set up the interface to external financial accounting programs, covering export/import paths, account numbers, and various options for data synchronization like open items (OP) and customer data. Subsequent sections cover 'Dokumente' (Documents), detailing document-specific options like model catalogs, document locking, and electronic declarations; 'Parameter' for general document processing rules; 'Preisberechnung' (Price Calculation) for setting up complex pricing rules; 'Lager/EK/EDI' (Warehouse/Purchasing/EDI) for inventory and purchasing settings; 'Archiv' for data archiving rules; 'Tagesabschluss' (End-of-Day) for data retention policies; 'System' for core system settings like measurement units and UI configurations; 'Kalkulation' (Calculation) for cost and markup settings; 'Druck' (Printing); and 'Produktion' (Production) for manufacturing-related configurations. The guide also includes information on managing employee data ('Mitarbeiter') and their access rights. It is intended for system administrators and advanced users responsible for the initial setup and ongoing maintenance of the A+W Business system."

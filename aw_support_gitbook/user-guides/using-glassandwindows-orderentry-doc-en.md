## 1. Using alternative functions with processing

### 1.1 Create order
Here only ~1000x1000 lites are entered without processings.

Start the alternatives function.

### 1.2 Add processing
Change mode to "Shapes/Processing" and press [Definition].

### 1.3 Enter the processing
Select the processing you want to use, in this case a polishing that I want on 4 sides the press [Execute].

### 1.4 Press Insert/Replace

### 1.5 Check tab 2 before executing change
To see what the result will be after the processings are added.

### 1.6 Press [Execute]
You are now done, the processings have been added to the positions on the order.

## 2. Using alternative functions to exchange articles

### 2.1 Create order
The purpose of this example is to exchange the first glass from float 5mm to another glass.

Start the alternatives function.

### 2.2 Add glass-position in mode
Change mode to that glass-position you want to change 1, 2 or 3. You can also change which glasses you want to change by unchecking the boxes on the left side of the articles.

### 2.3 Press Insert/Replace
Type in that article you want to replace in Insert/replace by or use the magnifying glass and then press Insert/replace.

### 2.4 Press [Execute]
You are now done, the articles have been replaced to the positions on the order. It's the same principle for mode "main product" where you change the whole main article etc.

## 3. Using alternative functions to change price
Create order in the way you want like in 2.1 and start alternative functions.

After you press execute the changes have been made.

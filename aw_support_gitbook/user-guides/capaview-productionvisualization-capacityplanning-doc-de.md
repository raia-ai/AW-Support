## A+W Capa View

A+W Capa View ist ein Service mit einer Web-basierten Oberfläche, um Daten aus A+W Production, anzuzeigen.

A+W Capa View kann in jedem Browser der neuesten Version gestartet werden. Dazu kann die Adresse auf den gewünschten Server als Favorit oder als Link auf dem Desktop hinterlegt werden – wie alle Internet-Adressen. Außerdem können Sie A+W Capa View direkt in A+W Business öffnen (Dokumentenerfassung > Funktionen > Kapazitätsübersicht). Für A+W Dashboard wurde ein neuer Anzeigentyp implementiert, in dem Sie die Anzeigen der A+W Capa View einbinden und darstellen können.

### Ziel des Programms

A+W Capa View ist ein Tool zur Visualisierung der verfügbaren Kapazitäten in der Produktion und des Fertigungsfortschritts. Es zeigt die Schichten, die in A+W Capacity Planning angelegt sind. Innerhalb der Schichten wird die Anzahl der unverplanten und verplanten Kapazitäten und der produzierten Aufträge angezeigt. Zudem haben Sie die Möglichkeit sich den auftrags- und laufbezogenen Produktionsfortschritt anzeigen zu lassen und können so Ihren Kunden immer aktuelle Auskünfte über die Aufträge geben.

*(Bild: Abb. 1 Monitor-Anzeige (Startbild) zeigt ein Balkendiagramm mit Produktionsdaten über mehrere Tage.)*

Zur Anzeige werden A+W Capacity Planning und die Daten der BDE herangezogen. Damit ermöglicht A+W Capa View z. B. Vertriebsmitarbeitern einen Blick auf die aktuellen Kapazitäten der Produktion. So können die Vertriebsmitarbeiter beim jeweiligen Kunden vor Ort eine Aussage treffen, ob ein großes Projekt, z. B. ein Bauvorhaben, noch in einer bestimmten Woche eingelastet werden kann.

Mitarbeiter der Auftragserfassung können sich Verknüpfungen mit voreingestellten Ansichten von Engpassbetriebsmitteln einrichten und so einen schnellen Überblick über die Produktionskapazitäten bekommen. Zudem können Sie sich mit einem Doppelklick auf eine Schicht den auftrags- und laufbezogenen Produktionsfortschritt ansehen.

> **Korrekte Darstellung nur mit korrekten BDE-Buchungen**
> Voraussetzung für aussagekräftige Anzeigen sind ein aktuelles A+W Capacity Planning und lückenlose BDE-Buchungen.

*(Bild: Abb. 2 Systemportfolio der A+W Software GmbH für den Bereich Flachglas zeigt die Integration von A+W Capa View in das gesamte Software-Ökosystem, das ERP, PPS, Maschinensteuerung und Controlling umfasst.)*

### Engpass-Betriebsmittel betrachten

A+W Capa View zeigt alle Maschinen an, die in A+W Capacity Planning definiert wurden.

Schauen Sie sich die Engpass-Betriebsmittel an, um zu entscheiden, ob ein Auftrag noch in der jeweiligen Woche produziert werden kann oder nicht. Die Engpass-Betriebsmittel bilden das Nadelöhr der Produktion und sind damit bei der Betrachtung kritischer als die anderen Betriebsmittel.

### Leistungsmerkmale

A+W Capa View zeigt die Kapazitäten der Betriebsmittel an und sorgt damit für hohe Transparenz.

A+W Capa View zeigt den auftrags- und laufbezogenen Produktionsfortschritt an. Sie können somit Ihren Kunden immer eine aktuelle Auskunft über Aufträge geben.

In Echtzeit erhalten Sie Auskunft über einzelne Betriebsmittel wie Zuschnitt, ISO-Linien, Bearbeitungen, ESG-Fertigung: Sie sehen die unverplanten und verplanten Kapazitäten und bereits produzierten Aufträge.

Das Programm kann sowohl auf großen Monitoren in den verschiedenen Abteilungen der Fertigung eingesetzt werden als auch auf mobilen Endgeräten wie Smartphones und Tablets. Voraussetzung ist ein Internet-Browser und ein Zugang zum Programm (Adresse, Benutzer-Account). Bei mobilen Endgeräten wird die Bildschirmanzeige beim Drehen des Geräts angepasst.

Damit dient A+W Capa View als Online-Monitoring-Tool und Entscheidungsmittel, in welcher Woche Aufträge geliefert werden können und ob entsprechende Zusagen an Kunden gemacht werden können. Oder es gibt Mitarbeitern aus der Arbeitsvorbereitung einen schnellen Überblick über Produktionskapazitäten.

In A+W Capa View können alle Maschinen angezeigt werden, die in A+W Capacity Planning definiert sind.

## Darstellung der Daten

Die Auswertungen von A+W Capacity Planning und den BDE-Buchungen werden in Balkendiagrammen visualisiert. Die Daten werden in der Monitor- und der Detail-Anzeige angezeigt.

### Monitor-Anzeige

Die Monitor-Anzeige ist der Startdialog von A+W Capa View. Sie können sich Daten zu Maschinen oder Maschinengruppen an bestimmten Tagen und Schichten anzeigen lassen.

*(Bild: Abb. 3 Balkendiagramm Monitor-Anzeige (Beispiel) zeigt einen einzelnen Tagesbalken mit mehreren Schichten.)*

*   **A** Zeitachse
*   **B** Schicht-Dauer (blau eingefärbt) in Stunden
*   **C** unverplante Kapazitäten einer Schicht
*   **D** verplante Kapazitäten einer Schicht
*   **E** bereits produzierte Aufträge einer Schicht
*   **F** Tag
*   **G** Schichten des angegebenen Tags

In diesem Beispiel werden zur gewählten Maschine drei Schichten und deren Auslastung angezeigt. Die Auslastung wird farblich unterschieden:
- **blau**: unverplante Aufträge/Positionen/Läufe
- **grau**: verplante Aufträge/Positionen/Läufe
- **grün**: bereits produzierte Aufträge/Positionen/Läufe

### Detail-Anzeige

Zu jeder Schicht kann die Detail-Anzeige geöffnet werden.

*(Bild: Abb. 4 Detail-Anzeige (Beispiel) zeigt eine detaillierte Ansicht einer einzelnen Schicht, aufgeteilt in einen oberen Übersichts- und einen unteren Detailbereich.)*

*   **A** Übersicht zur Schicht
*   **B** Übersicht zu Läufen oder Aufträgen

*(Bild: Abb. 5 Balkendiagramm Detail-Anzeige, oberer Bereich zeigt drei Balken, die Zeit, Quadratmeter und Stückzahlen repräsentieren.)*

*   **A** Schichtzeit in Stunden
*   **B** Fläche in Quadratmetern
*   **C** Stückzahlen
*   **D** produzierter Anteil der Schicht
*   **E** verplanter Anteil der Schicht
*   **F** unverplanter Anteil der Schicht

Im oberen Teil der Detailanzeige werden zur gewählten Schicht drei Balken angezeigt.
- Verplante und unverplante Zeiten der Schicht und Zeiten, in denen produziert wurde.
- Verplante, unverplante und produzierte Flächen der Schicht, Einheit: Quadratmeter.
- Verplante, unverplante und produzierte Teile der Schicht, Einheit: Stück.

Die Balken werden immer über die ganze Breite des Fensters angezeigt, was der gesamten Zeit, Fläche oder Stück der jeweiligen Schicht entspricht.

*(Bild: Abb. 6 Balkendiagramm Detail-Anzeige, unterer Bereich zeigt eine Liste von Läufen oder Aufträgen mit individuellen Fortschrittsbalken.)*

Für den unteren Teil der Detailanzeige wählen Sie, ob die Daten lauf- oder auftragsbezogen angezeigt werden. Sie können wählen, nach welchen Kriterien die Daten angezeigt werden:
- Zeit in Stunden
- Stück
- Menge in Quadratmetern

Der Mitarbeiter kann sehen, wieviel von den jeweiligen Aufträgen schon produziert wurde. Im Laufe einer Schicht wandeln sich die grauen Balken, also verplante Kapazitäten, hin zu grünen Balken, also bereits produzierten Kapazitäten.

> **Korrekte Darstellung nur mit korrekten BDE-Buchungen**
> Voraussetzung für eine aussagekräftige Detailanzeige sind lückenlose BDE-Buchungen.

### Monitor-Anzeige

Auf der Startseite (Monitor-Anzeige) werden die Daten zu Maschinen oder Maschinengruppen schichtbezogen angezeigt.

*(Bild: Abb. 7 Schaltflächen in der Monitor-Anzeige zeigt die Kopfzeile der Anwendung mit verschiedenen Filtern und Steuerelementen.)*

*   **A** Sortierung nach Schichten oder Tagen
*   **B** Sortierung nach Maschinen oder Maschinengruppen
*   **C** Auswahlliste Maschinen oder Maschinengruppen
*   **D** Sortierung nach Zeit, Stück oder Menge (Quadratmeter)
*   **E** Starttag des Zeitraums
*   **F** Wahl des Datums
*   **G** Zeitraum der Auswertung
*   **H** Anzeige verschieben um 7, 10 oder 14 Tage
*   **I** Aktualisiert die Anzeige
*   **J** Öffnet den Dialog Einstellungen

Die folgenden Funktionen gelten bei der Monitor-Anzeige in A+W Capa View.

| Bedienung | Beschreibung |
| :--- | :--- |
| **Sortierung nach Schichten oder Tagen (A)** | Zeigt die Balkendiagramme in Schichten oder Tagen an. |
| **Sortiert nach Maschinen oder Maschinengruppen (B)** | Zeigt die Balkendiagramme in Maschinen oder Maschinengruppen an. |
| **Auswahlliste Maschinen oder Maschinengruppen (C)** | In der Auswahlliste stehen die Maschinen oder Maschinengruppen zur Auswahl, die in A+W Production definiert wurden. Es ist sinnvoll, in A+W Production Maschinengruppen, z. B. je Produktionsbereich, zu erstellen. So können Sie in A+W Capa View alle kritischen Stellen der Produktion auf einen Blick einsehen. |
| **Sortierung nach Zeit, Stück oder Fläche (D)** | Sie können die Balkendiagramme nach folgenden Inhalten anzeigen lassen: Zeit in Stunden, Stück, Menge in Quadratmetern. |
| **Starttag des Zeitraum (E)** | Starttag, den Sie im Kalenderfeld gewählt haben. |
| **Wahl des Datums (F)** | Öffnet das Kalenderfeld, in dem Sie einen Starttag wählen. |
| **Zeitraum der Auswertung (G)** | Sie können einen Zeitrahmen von 7, 10 oder 14 Tagen anzeigen lassen. |
| **Anzeige verschieben um 7, 10 oder 14 Tage (H)** | Verschiebt die Anzeige der Daten um 7, 10 oder 14 Tage vorwärts oder rückwärts. |
| **Aktualisiert die Anzeige (I)** | Aktualisiert die Anzeige. |
| **Öffnet den Dialog Einstellungen (J)** | ⇨ "Einstellungen" auf Seite 17. |

### URLs

> **Hilfe öffnen**
> So öffnen Sie die Hilfe: Rechtsklick auf den Bildschirm, dann die *Einstellungen* öffnen, dort finden Sie die Option *Hilfe*.

Sie können A+W Capa View mit verschiedenen Geräten und aus verschiedenen Anwendungen heraus öffnen. Sie können die URL einer Anzeige im Dialog *Einstellungen* kopieren. Mit einer kopierten URL werden die aktuellen Einstellungen übernommen. Damit haben Sie auf allen Geräten z. B. das gleiche Farbschema, die gleiche Maschine oder den gleichen Zeitraum.

#### URLs als Favoriten und Verknüpfungen

Kopieren Sie die URL mit den aktuellen Einstellungen und speichern Sie diese im Browser als Favorit ab, oder Sie erstellen eine Verknüpfung auf dem Desktop. So können Sie sich Daten zu bestimmten Maschinen, Maschinengruppen oder Schichten voreingestellt anzeigen lassen.

#### Eingebunden in A+W Dashboard

Sie können die URL der aktuellen Anzeige in A+W Dashboard anzeigen lassen. Sie haben so voreingestellte Daten schnell im Blick.

#### Eingebunden in A+W Business

Die Daten von A+W Capa View lassen sich aus A+W Business heraus über folgenden Pfad öffnen:
*Dokumente Angebot > Angebot > Funktionen > Kapazitätsübersicht*.

### Einstellungen

Die Einstellungen können aus jeder Anzeige heraus geöffnet werden. Machen Sie die Einstellungen, bevor Sie Daten von Maschinen anzeigen lassen, da die Anzeige beim Ändern der Einstellungen auf die Default-Werte gesetzt werden.

*(Bild: Abb. 9 Einstellungen der Bedienoberfläche zeigt ein Dialogfenster mit Optionen für Mandant, Sprache, Schema und URL-Abruf.)*

*   **A** Dialog Einstellungen schließen
*   **B** Mandant
*   **C** Sprache
*   **D** Farbschema
*   **E** URL abrufen
*   **F** Hilfe aufrufen

| Funktion | Bedienung |
| :--- | :--- |
| **Dialog Einstellungen schließen (A)** | Schließt den Dialog *Einstellungen*. |
| **Mandant (B)** | Wenn Sie verschiedene Produktionsstandorte haben: Wählen Sie zwischen den Mandanten. |
| **Sprache (C)** | Wählen Sie Ihre Sprache. |
| **Farbschema (D)** | Mit der Einstellung des Farbschemas können Sie die Darstellung an die Umgebung anpassen, in der der Monitor steht. Ein helles Farbschema eignet sich z. B. für Orte mit schlechteren Lichtverhältnissen. |
| **URL abrufen (E)** | Öffnet ein Popup-Fenster, in dem Sie die URL von A+W Capa View kopieren können.<ul><li>Sie können mit der URL ein Lesezeichen im Browser einrichten.</li><li>Sie können die kopierte URL versenden, um sie an weiteren Geräten nutzen zu können.</li><li>Zudem können Sie die Daten mit der URL in A+W Dashboard importieren und dort eine Anzeige erstellen.</li></ul> |
| **Hilfe aufrufen (F)** | Öffnet die Hilfe. |

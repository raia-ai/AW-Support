long_description: "This document serves as a comprehensive user manual and software reference for the A+W Business Pro system, with a specific focus on the Stock Management and Capacity Planning modules. It is designed for users who manage inventory, purchasing, and production schedules within the glass, window, and door industries. The first part of the document covers stock management functionalities, including how to use the 'Future stock on hand' tab to check inventory levels, reservations, and incoming purchase orders. It explains how to manage a 'Stock P.O.' (Purchase Order) pool, detailing menu options for transferring suggested P.O.s to purchasing, handling order-related transfers, and checking delivery dates. Furthermore, it provides an in-depth guide to 'Inventory Assessment,' explaining the periodic processes for LIFO (Last-In, First-Out) and FIFO (First-In, First-Out) valuation, including calculation examples and procedures for handling initial assessments. The second major part of the document is a tutorial on 'Capacity Planning.' It outlines the basic principles, master data setup, and modular structure. Key topics include the configuration of machines, production areas, work types, work units, and shift calendars. It details how to define default times, special times, time surcharges, and transition times to accurately calculate production capacity and costs. The manual explains the interplay between master data components to enable effective scheduling, detection of capacity bottlenecks, and monitoring of production status. It includes step-by-step instructions, examples, and exercises for setting up and using the system."

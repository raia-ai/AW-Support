# Kompletterfassung von Dokumenten bis Faktura

---
## Kreditlimit

Sie können ein Kreditlimit festlegen und automatisch prüfen lassen. Im Zusammenspiel mit einem Programm zur Finanzbuchhaltung (FiBu), das offene Posten zurückmeldet, erhalten Sie einen aussagekräftigen Überblick über den aktuellen Finanzstatus eines Kunden. Diese Funktion ist im Part Stammdaten beschrieben.

*Abb. C-9 Kreditlimit des Kunden*

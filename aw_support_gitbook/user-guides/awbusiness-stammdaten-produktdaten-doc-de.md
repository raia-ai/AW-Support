# Tutorial 1: Produktdaten

---
## Lagermaße

Zu einem Produkt können Sie mehrere Lagermaße mit unterschiedlichen Maßen anlegen.

Die Beschaffungsart für Gläser mit definierten Größen ist in der Regel Lagerentnahme. Sie können jedoch auch Lagermaße (für die Preisfindung) anlegen, die nicht im Lager geführt werden.

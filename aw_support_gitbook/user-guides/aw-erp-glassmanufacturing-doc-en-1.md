## Overview

### Features
- Complete ERP-system
- Control of company areas
- Connection to A+W Production, A+W Logistics and external software
- Master data management, quotation creation, order processing
- Graphic interface for easy use
- Flexible price calculation options

### Specs
- **Operating System**: MS Windows (see A+W Clarity Hardware Recommendation)
- **Data Base**: SQL Server
- **Reports**: Crystal Reports or QLIK

## Sales

### Features
- Graphical order entry
- Real commercial and technical build of material
- Detailed document management possibilities
- Delivery date control & statistics
- Standardized processing catalog

### Specs
- Master data for customers, products and prices needs to be setup

## Purchasing

### Features
- Automated creation of purchase orders from orders
- Supplier inquiries
- Definition of conditions and prices, per supplier
- Receipt of goods booking with missing quantities control, Invoice control, statistics

### Specs
- Master data for suppliers and allocation of products needs to be setup

## Stock Management

### Features
- Stock bookings & stock movements
- Future stock on hand
- Interconnection with Purchasing and Goods Receipt and Production
- Different evaluation of stock (FIFO, LIFO or weighted moving average)
- Stock orders (filler)

### Specs
- Management of different jumbo sizes, stock sizes, residue plates, cases, glass doors, fittings, frame parts and accessories, such as mountings, sealing tape, etc.

## Georgian Bar Entry

### Features
- Design grill patterns for 1st and 2nd cavity
- Creation of technical drawings
- Complex Construction capabilities
- Different colors and variants

### Specs
- Master data for bars needs to be setup

## Shape & Processing Entry

### Features
- Creation of technical drawings
- Standard Processing and Shape Catalog
- Graphical display & zoom functionality
- Technical Order Entry

### Specs
- Possible addons includes CAD Designer, DXF Import, iTOE, Dorma
- Additional functions are Macros, Templates depending on addons

## Dispatch Control

### Features
- Status control and overview (e.g. finished, complete, in time, ready for shipping)
- Route planning
- List and form printing (e.g. delivery note)
- Consignment
- Dispatch control

## Controlling

### Features
- Price recorder
- Order/P.O. history
- Connected to A+W Production with status updates
- Workflow and status management for automized procedures
- Cost calculation
- Credit limit control

### Specs
- For full cost calculation, A+W Production with Capacity planer is needed

## Statistics

### Features
- Overview of the commercial progress
- Important instrument for analyzing the company
- Possibility of customized reports (e.g., Crystal Reports or QLIK)
- Deep and customized pivot statistics (MS Excel export)

### Specs
- Volume & consumption statistics
- Turnover, supplier or delivery statistics

## Object Management

### Features
- Project based prices and conditions
- Max. quantities (qty, sqm, pieces)
- Definition of a validity date range
- (Re-) Calculation of orders / quotes during entry
- Project overview in order entry

### Specs
- Object Master data needs to be setup

## Product Regulation – Conformity Pack I+II

### Features
- Integration/Interaction with Sommer Informatik Sommer Global
- Calculation of technical Specs
- Creation of Declaration of performance

### Specs
- 2 Variants – Interaction with each client/Interaction with Master data client
- Needs external license of Sommer Global

## A+W Excel Line Item Import

### Features
- This module enables the item import from an Excel file.
- Various Excel formats can be imported.
- The routing which Excel column contains which information can be defined interactively during import or can be pre-defined per customer.
- Allows import of items based on an already existing item with the addition of particular properties (like height and width) via the clipboard.
- To start the functionality, an existing item is marked in item entry and the "Import from clipboard" function is selected via context menu.

## A+W Door Order Entry: Fitting and Processing

### Features
- Simplifies the entry of glass door fittings and processings.
- The appropriate processing's can be assigned to the fittings in the master data.
- When exchanging or adding fittings, the respective processing's are automatically exchanged or supplemented.

### Specs
- Fittings are part of the BOM (Bill of Materials).

## A+W Capa View

### Features
- Keep a handle on dates – both on the road and in the office.
- Quick overview of machine utilization for the next few days and weeks.
- Complete deliveries on the date defined thanks to online production overview of orders and batches.
- Conduct date negotiations with customers and sales employees on the road and regardless of which device you are using.

## A+W iTOE (Integrated Technical Order Entry)

### Features
- Integrated Technical Order Entry with multiple uses.
- Import of DXF or SN files in order entry.
- Synchronization of processing's between order entry and A+W CAD Designer (Shapes).
- Entry of standard geometries and processing's in A+W order entry.
- Generation of geometric data (SN files) from the A+W ERP systems' order data.
- Import of geometric data from the A+W CAD Designer (Shapes) into order entry with automatic generation of the BOM.
- Integrated shape detection can convert free shapes back to standard shapes.

### Specs
- Microsoft SQL Server 2016 required.

## A+W CAD Designer - Bars

### Features
- Quick and easy creation and dimensioning of complex bar designs.
- Intuitive graphical user interface, which allows the entry of constructions such as sun, moon, etc.
- Direct control of the bar saw possible.
- Cost and time savings through the generation and use of template files and integrated logic for bar distribution (e.g. same opening areas or same drill point distance).

### Specs
- No additional master data needed.

## A+W CAD Designer - Shapes

### Features
- Quick and easy creation of complex individual shapes and standard template files for re-use in ERP systems.
- Easy entry and modification of difficult shapes and processing (e.g. edge deletion or complex cut-outs).
- Possibility of direct control of your CNC machine via DXF transfer.
- Extension of your product-specific shape catalog with free constructions.
- Import and processing of DXF-files.

## A+W CX Stock Forecast

### Features
- System automatically highlights articles with shortage over a given period.
- Evaluate consumption based on last year's average consumption.
- Forecasting orders possible without releasing them into production.
- Incoming purchase orders automatically evaluated.

### Specs
- A+W Enterprise Stock must be configurated accordingly.

## A+W EDI Import

### Features
- Thanks to the external EDI interface, customer orders can be automatically imported into the ordering system.

### Specs
- Customer's or supplier's ERP system must be able to read and populate A+W's EDI interface.

## A+W Gantry Interface

### Features
- Synchronization of stock balances of A+W ERP Inventory systems with Hegla or Bystronic inventory systems.
- Stock entries and exits are exchanged, and stock balances can be automatically aligned on a periodic basis.
- Up-to-date inventory overview in the portal storage within the ERP stock system.
- Online connection enables continuously integrated processes from ordering to consumption of glass in production.
- No time-consuming manual inventory or stock correction in the ERP system is required.

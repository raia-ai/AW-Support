long_description: "This document is a sales and marketing presentation detailing key add-on modules for the A+W software ecosystem, which is targeted at the glass, window, and door manufacturing industries. It provides a high-level overview of the features and specifications for five distinct modules. 'Statistics/BI' covers data evaluation, reporting, and KPI visualization. 'Extended Workbench' outlines the process for managing external processing steps, including transport and goods receipt, within a single production order. 'Product Regulation – Conformity Pack I+II' describes the integration with Sommer Informatik to handle technical specifications and create Declarations of Performance for regulatory compliance. 'A+W Capa View' offers a tool for real-time capacity and machine usage monitoring, aiding in delivery date negotiation. Finally, 'A+W CX Dispatch' presents a modern graphical interface for managing dispatches, identifying critical items, and tracking production status. The document is designed to showcase the capabilities and benefits of these modules to potential and existing customers."

## Statistics/BI

### Features
- Statistical evaluations based on the A+W ERP- und PPS- data bases – live and archive db's
- Full Control over all aspects of analysis apps, permissions and data management
- Scalable data models and applications
- Usage of predefined A+W reports
- Creation of self-defined reports

### Specs
- High information density
- Graphical display of KPI's
- Adjustable user interface

*(Image displays a dashboard with a data table showing customer statistics like order amount and margin, alongside a pie chart and a line graph for visualizing data.)*

## Extended Workbench

### Features
- Triggering of a properly scheduled purchase order
- Organization of the transport out and back via dispatch control or direct to the end-customer
- Goods receipt in A+W Enterprise and A+W Production (continuation of production)
- Control with a single order in production, no sub-orders necessary

*(Image displays a workflow diagram illustrating an internal production process (Cut, Grind) and an external process (Print), followed by a final internal step (IGU) before dispatch to the customer.)*

---

## Product Regulation – Conformity Pack I+II

### Features
- Integration/Interaction with Sommer Informatik Sommer Global
- Calculation of technical specifications
- Creation of Declaration of performance

### Specs
- 2 Variants – Interaction with each client or one Master data with data distribution
- External license from Sommer Global required

*(Images display the A+W Enterprise order entry screen showing product details, including a Declaration of Performance (DoP) number, and the SommerGlobal software interface for managing product data and projects.)*

## A+W Capa View

### Features
- Keep a handle on dates – both on the road and in the office
- Quick overview of machine usage for the coming days and weeks
- Complete deliveries on time thanks to online production overview of orders and batches
- Conduct date negotiations with customers and sales employees on the road and regardless of which device you are using

*(Image displays the A+W CapaView interface, which shows a bar chart representing machine capacity and usage over a period of days.)*

---

## A+W CX Dispatch

### Features
- Modern graphical interface allows intuitive work with user self-defined views
- Configurable workflow to identify critical items (excessive length, weight, models, ..)
- Missing quantity function
- Displaying A+W Production status from BDE scanning and last scanned machine

*(Image displays the A+W Clarity Suite for dispatch, showing an overview of a partial delivery with details on items, quantities, and delivery status.)*

long_description: "This comprehensive software reference guide details the functionality of the A+W Business Pro Capacity Planning module. It is intended for users responsible for production scheduling and management. The manual is divided into major sections covering 'Einlastung' (Scheduling), 'Produktionsmeldungen' (Production Reports), and 'Leitstand' (Control Center). The 'Einlastung' section explains how to change scheduling criteria, determine manufacturing deadlines, manage machine aggregates, and handle capacity problems. It covers various dialogs for scheduling orders, including options for automatic and manual capacity handling. The 'Produktionsmeldungen' section describes how to manually report the completion status of orders, positions, or individual parts at different production stations, which is crucial when not using automated data collection (BDE). It includes instructions for creating production lists and viewing graphical representations of production progress. The 'Leitstand' section provides an overview of the entire production schedule, allowing users to monitor and correct the loading of all machines. It explains how to handle machine downtimes, re-schedule affected orders, and analyze capacity utilization across different timeframes, machines, and production areas. The document includes navigation paths, dialog box descriptions with all available options, and explanations for various settings to ensure efficient and accurate production planning."

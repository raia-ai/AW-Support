# Produktionsmeldungen

Der aktuelle Stand der Fertigung kann in die Kapazitätsplanung übernommen werden, damit neben dem Status der Aufträge auch Übersichten erstellt werden können, z. B. Produktionslisten und grafische Darstellungen zum Fertigungsstand.

Wenn Sie nicht mit Rückmeldungen aus der Betriebsdatenerfassung (BDE) arbeiten, können Sie in den Dialogen zur manuellen Fertigmeldung an den verschiedenen Stationen der Produktion Läufe, Aufträge oder Auftragspositionen fertig melden.

In den Dialogen können Sie sich alle Aufträge anzeigen lassen oder nur die Aufträge, die noch nicht fertig gemeldet wurden. Sie können sowohl Fertigmeldungen als auch Bruchmeldungen erfassen.

Die Rechte zum fertig melden können in gesonderten Dialogen vom Administrator eingeschränkt werden.

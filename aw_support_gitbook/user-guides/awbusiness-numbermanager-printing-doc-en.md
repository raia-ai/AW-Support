# Number Manager

---
## Route optimization group

This group is only enabled if the A+W Logistics Optimizer is installed on your system.

-   **Transfer number manager**:
    Transfer orders from the number manager in the delivery mode. The order numbers are sent to the logistic service, which reads the required data from the database.

## Number Manager

**Documents > Order > NM order**

The Number manager dialog and the corresponding menus are the same for all document types and modules. We are going to describe it in connection with an Order in this manual.

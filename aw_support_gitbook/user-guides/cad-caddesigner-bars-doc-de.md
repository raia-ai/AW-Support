### Sprossen anlegen (Bar 01-003)

**Ziel der Handlung**
Einen Sprossenartikel anlegen.

**Voraussetzungen**
-

**Zusatzinfo**
-

**Workflow**
1. Öffnen Sie den Dialog **Stammdaten-Editor** [Stammdaten > Stammdaten-Editor].
2. Klicken Sie auf die Schaltfläche [Hinzufügen]. Geben Sie die gewünschte Artikel-Nummer ein.
3. Geben Sie im Feld **Breite** die Sprossenbreite ein.
4. Geben Sie im Feld **Frästiefe** (verstellbare Begrenzungsanschlag, gegen den das vorher gekappte Sprossenende positioniert wird) den entsprechenden Wert ein.
5. Geben Sie im Feld **Randabzug** (Stärke des Kunststoff-Stopfen) den entsprechenden Wert ein.
6. Wenn gewünscht, aktivieren Sie die Checkbox **Spezial-Kreuzung** (keine durchgehende Sprosse - Steckverbindungen) und geben im Feld **Breite Kreuz** den entsprechenden Wert ein.
7. Geben Sie im Feld **Stablänge** die Länge des Sprossenstabes ein.
8. Geben Sie im Feld **T-Zugabe** einen Wert ein, wenn die Sprosse im rechten Winkel auf eine andere Sprosse zugeschnitten wird.
9. Die Radiotasten im Bereich **Durchgang** dienen der Festlegung von Sprossendurchgängen, die von außen mittels Übergabedateien an A+W CAD Designer übergeben werden.
10. Aktivieren Sie die Checkbox **Doppelter Zuschnitt**, wenn für den verwendeten Sprossentyp die Zuschnittmenge automatisch verdoppelt werden soll (3fach ISO).
11. Wenn Sie die Checkbox **L-Felder mit Abzug** aktivieren, werden die Randstopfen der Sprosse nicht mit vermaßt.
12. Bei mehreren Sprossenfertigungslinien können Sie im Feld **Liniennummer** die entsprechende Nummer eingeben.
13. Im Feld **Farbcode** weißen Sie der Sprosse die RAL-Farbe zu.
14. Handelt es sich bei der Sprosse um die Standard-Sprosse, klicken Sie auf die Schaltflächen [Start].

### Stammdaten schützen (Bar 01-004)

**Ziel der Handlung**
Die Stammdaten mit einem Passwort schützen.

**Voraussetzungen**
Stammdaten sind angelegt.

**Zusatzinfo**
-

**Workflow**
1. Öffnen Sie den Dialog **Stammdaten-Editor** [Stammdaten > Stammdaten-Editor].
2. Aktivieren Sie die Checkbox **Passwortschutz** im Bereich **Optionen**.
3. Klicken Sie auf die Schaltfläche [OK]. Es folgt eine Abfrage, ob die Stammdaten gesichert werden sollen.
4. Klicken Sie auf die Schaltfläche [Ja]. Es öffnet sich der Dialog **Passwort?**
5. Geben Sie im Feld **Passwort** das gewünschte Passwort ein.
6. Wiederholen Sie im Feld **Bestätigung** das eingegebene Passwort.
7. Klicken Sie auf die Schaltfläche [OK].
8. Wenn Sie das nächste Mal den Stammdaten-Editor öffnen, werden Sie nach dem Passwort gefragt.

## Sprossen

### Eigenschaften der Scheibe (Bar 02-001)

**Ziel der Handlung**
Die Eigenschaften der Scheibe definieren.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.

**Zusatzinfo**
-

**Workflow**
1. Klicken Sie auf die Symbol-Schaltfläche [Neue Scheibe]. Der Dialog **Modell-Erfassung** öffnet sich.
2. Wählen Sie die Modell-Nr. aus und geben Sie die Maße ein.
3. Anschließend öffnet sich der Dialog **Eigenschaften Scheibe**. Über die Felder **X-** und **Y-Position** können Sie die Position auf Ihrer Arbeitsfläche bestimmen. Zum Drehen der Scheibe dient das Feld **Ausrichtung**.
4. Die Felder im unteren Bereich des Dialogs dienen der Eingabe der Auftragsdaten. Füllen Sie diese entsprechend aus.
5. Im Bereich **Extra** haben Sie die Möglichkeit für diverse Einträge (z. B. Liefertermin, Abstandhalter, Gaskürzel, etc.) Texte zu hinterlegen. Im Falle einer A+W Production-Anbindung, werden diese Felder bei der Übergabe automatisch gefüllt.
6. Über die Schaltfläche [OK] verlassen Sie den Dialog. Die Scheibe wird platziert.

### Einfaches Sprossenmuster platzieren (Bar 02-002)

**Ziel der Handlung**
Zwei senkrechte und zwei waagerechte Sprossen platzieren.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.
- Scheibe muss platziert sein.

**Zusatzinfo**
-

**Workflow**
1. Klicken Sie auf die Symbol-Schaltfläche [Neues senkrecht/waagerecht Muster] und anschließend auf die Scheibe. Der Dialog **Muster Senkrecht-Waagerecht** öffnet sich.
2. Geben Sie die Anzahl der Sprossen ein.
3. Aktivieren Sie im Bereich **Teilung Waagerecht** die Option **Gleiche lichte Felder**.
4. Aktivieren Sie im Bereich **Teilung Senkrecht** die Option **Gleiche lichte Felder**.
5. Aktivieren Sie im Bereich **Durchgängige Sprossen** die Option **Senkrechte**.
6. Über die Schaltfläche [OK] verlassen Sie den Dialog. Die Sprossen werden platziert.

---

### Sprossen über Werte platzieren (Bar 02-003)

**Ziel der Handlung**
Sprossen über feste Werte platzieren.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.
- Scheibe muss platziert sein.

**Zusatzinfo**
-

**Workflow**
1. Klicken Sie auf die Symbol-Schaltfläche [Neues senkrecht/waagerecht Muster] und anschließend auf die Scheibe. Der Dialog **Muster Senkrecht-Waagerecht** öffnet sich.
2. Geben Sie in den Feldern **Anzahl Waagerecht** und **Anzahl Senkrecht** jeweils eine 2 ein. Aktivieren Sie in den Bereichen **Teilung Waagerecht** und **Teilung Senkrecht** die Option **Kundendefiniert**. Klicken Sie auf die Schaltfläche [OK]. Es öffnet sich der Dialog **Kundenvorgabe SW-Muster**.
3. Aktivieren Sie im Bereich **Waagerechte Sprosse** die Option **Scheibenrand**. Aktivieren Sie die Radiotaste **Waager.**. Klicken Sie auf die Schaltfläche [Hinzufügen]. Es öffnet sich der Dialog **Eingabe Sprossen-Abstände**. Geben Sie **150 mm** ein. Klicken Sie auf [OK]. Der Dialog wird geschlossen. Wiederholen Sie den Vorgang mit **300 mm**. Aktivieren Sie die Radiotaste **Von unten**.
4. Aktivieren Sie im Bereich **Senkrechte Sprosse** die Option **Scheibenrand**. Aktivieren Sie die Radiotaste **Senkr.**. Klicken Sie auf die Schaltfläche [Hinzufügen]. Es öffnet sich der Dialog **Eingabe Sprossen-Abstände**. Geben Sie **200 mm** ein. Klicken Sie auf [OK]. Der Dialog wird geschlossen. Wiederholen Sie den Vorgang mit **400 mm**. Aktivieren Sie die Radiotaste **Von links**. Über die Schaltfläche [OK] verlassen Sie den Dialog.
5. Aktivieren Sie im Dialog **Muster Senkrecht-Waagerecht** im Bereich **Durchgängige Sprossen** die Option **Senkrechte**.
6. Über die Schaltfläche [OK] verlassen Sie den Dialog. Die Sprossen werden platziert.

### Modell mit Sprossen platzieren (Bar 02-004)

**Ziel der Handlung**
Ein Modell mit Sprossen platzieren.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.

**Zusatzinfo**
-

**Workflow**
1. Klicken Sie auf die Symbol-Schaltfläche [Neue Scheibe] und anschließend auf die Arbeitsfläche. Der Dialog **Modell-Erfassung** öffnet sich.
2. Öffnen Sie die Kombobox **Modell-Nummer** und wählen Sie das Modell **065** aus. Das Modell hat eine Höhe von **800** und eine Breite von **600**. Klicken Sie auf die Schaltfläche [OK]. Das Modell wird platziert und der Dialog **Eigenschaften Scheibe** öffnet sich.
3. Füllen Sie die Felder entsprechend aus und klicken Sie auf [OK].
4. Klicken Sie auf die Symbol-Schaltfläche [Neues senkrecht/waagerecht Muster] und anschließend auf das Modell. Der Dialog **Muster Senkrecht-Waagerecht** öffnet sich.
5. Geben Sie in dem Feld **Anzahl Waagerecht** eine 2 und in dem Feld **Anzahl Senkrecht** eine 1 ein.
6. Aktivieren Sie in den Bereichen **Teilung Waagerecht** und **Teilung Senkrecht** jeweils die Option **Gleiche lichte Felder**.
7. Aktivieren Sie im Bereich **Durchgängige Sprosse** die Option **Senkrechte**. Klicken Sie auf die Schaltfläche [OK].
8. Die Sprossen werden platziert.

### Freie Sprossenkonstruktion erstellen (Bar 02-005)

**Ziel der Handlung**
Eine freie Sprossenkonstruktion erstellen.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.

**Zusatzinfo**
-

**Workflow**
1. Die Schritte 1 bis 3 entnehmen Sie der Übung Freie Sprossenkonstruktion erstellen (Abmessungen: Breite=500, Höhe=600).
2. Klicken Sie auf die Symbol-Schaltfläche [Neues senkrecht/waagerecht Muster] und anschließend auf das Modell. Der Dialog **Muster Senkrecht-Waagerecht** öffnet sich. Geben Sie in dem Feld **Anzahl Waagerecht** eine 3 und in dem Feld **Anzahl Senkrecht** eine 2 ein.
3. Aktivieren Sie im Bereich **Teilung Waagerecht** die Radiotaste **Kundendefiniert**. Aktivieren Sie im Bereich **Durchgängige Sprossen** die Checkbox **Waagerechte**. Aktivieren Sie im Bereich **Teilung Senkrecht** die Radiotaste **Gleiche lichte Felder**. Klicken Sie auf [OK]. Es öffnet sich der Dialog **Kundenvorgabe SW-Muster**. Aktivieren Sie im Bereich **Waagerechte Sprossen** die Checkbox **Aufsummieren**. Fügen Sie drei Sprossen mit einem Abstand von 120 ein. Betätigen Sie die Schaltfläche [OK].
4. Setzen Sie auf die oberen Teile der senkrechten Sprossen jeweils drei Fangpunkte und verbinden Sie die mittleren Fangpunkte mit einer waagerechten Sprosse.
5. Löschen Sie die Fangpunkte und aktivieren Sie das Werkzeug **Sprosse setzen/löschen**. Markieren Sie auf der linken Seite die senkrechte Sprosse dort, wo sie abgeschnitten werden soll. Das obere Stück der senkrechten Sprosse wird gelöscht. Wiederholen Sie den Vorgang für die rechte Seite.
6. Aktivieren Sie das Werkzeug **Sonne**. Klicken Sie in das Feld, in dem die Sonne eingefügt werden soll. Es öffnet sich der Dialog **Sonne und Sterne**. Aktivieren Sie die Radiotaste **Stern** sowie die Checkbox **Gemeinsamer Zuschnitt**. Das Strahlenmuster hat gleiche Winkel und drei Strahlen.
7. Klicken Sie auf die Schaltfläche [OK]. Der Dialog wird geschlossen und das Strahlenmuster eingefügt.

## Spezialmuster

### Fassadenanlage erstellen (Bar 03-001)

**Ziel der Handlung**
Eine Fassadenanlage platzieren.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.

**Zusatzinfo**
-

**Workflow**
1. Wählen Sie im Menü **Makro > Spezial Muster > Fassadenanlage**. Es öffnet sich der Dialog **Fassadenanlage...**
2. Geben Sie folgende Scheibenabmessungen ein:
    - Scheibe A und D: Breite=800, Höhe=600
    - Scheibe B: Breite=400, Höhe=300
    - Scheibe C: Breite=200, Höhe=100
    Die vom Programm ermittelten Abstände werden übernommen.
    Betätigen Sie die Schaltfläche [OK]. Es öffnet sich der Dialog **Eigenschaften Scheibe**. Nehmen Sie die entsprechenden Eintragungen vor.
3. Klicken Sie auf die Schaltfläche [OK]. Der Dialog wird geschlossen und die Fassade platziert.

### Makro laden (Bar 03-002)

**Ziel der Handlung**
Ein vorhandenes Makro laden.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.

**Zusatzinfo**
-

**Workflow**
1. Platzieren Sie Modell-Nr. 0 mit einer Breite=800 und einer Höhe=600.
2. Füllen Sie den Dialog **Eigenschaften Scheibe**. Markieren Sie im Bereich **Extra** den Eintrag **Kommisionstext** und geben Sie **Wohnanlage II** ein.
3. Betätigen Sie die Symbol-Schaltfläche [Neues Makro]. Platzieren Sie den Mauszeiger auf dem Modell und betätigen Sie die linke Maustaste. Es öffnet sich der Dialog **Makro auswählen**.
4. Wählen Sie das Marko **York3H1S** aus und klicken Sie auf [OK].
5. Es öffnet sich der Dialog **Makro-Parameter**. Markieren Sie **Breite der Raute** und betätigen Sie die Schaltfläche [Ändern...].
6. Es öffnet sich der Dialog **Relativer Abstand**. Geben Sie für die Breite der Rauten **125 mm** ein. Schließen Sie den Dialog mit der Schaltfläche [OK].
7. Markieren Sie **Höhe der Raute** und klicken Sie auf die Schaltfläche [Ändern...].
8. Es öffnet sich der Dialog **Relativer Abstand**. Geben Sie für die Höhe der Rauten **150 mm** ein. Schließen Sie den Dialog mit der Schaltfläche [OK].
9. Schließen Sie den Dialog **Makro-Parameter** mit der Schaltfläche [OK].
10. Das Makro wird platziert.

**Ziel der Handlung**
Ein vorhandenes Makro abändern und speichern.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.

**Zusatzinfo**
-

**Workflow**
1. Platzieren Sie das Makro **York3H1S** wie in der Übung **Makro laden** beschrieben.
2. Aktivieren Sie das Werkzeug **Element auswählen**.
3. Markieren Sie die obere Sprosse der mittleren Raute. Drücken Sie die <Entf>-Taste auf Ihrer Tastatur.
4. Wiederholen Sie den Vorgang mit der unteren Sprosse der mittleren Raute.
5. Menü **Datei** öffnen. Menüpunkt **Speichern unter** auswählen.
6. Im Feld **Dateiname** **York3H1S-WAll** eingeben und den Dialog mit der Schaltfläche [OK] schließen.

### Eigenes Makro erstellen (Bar 03-004)

**Ziel der Handlung**
Ein eigenes Makro erstellen.

**Voraussetzungen**
- Stammdaten müssen vorhanden sein.

**Zusatzinfo**
-

**Workflow**
1. Öffnen Sie im Menü **Makro** den Menüpunkt **Neues Makro**. Auf Ihrem Bildschirm erscheint eine Scheibe. Die Scheibe hat die Maße 3 m x 3 m und hat 8 Fangpunkte (je 4 am Innenrahmen und 4 an der Scheibenaußenkante).
2. Setzen Sie jetzt je einen Fangpunkt (Intervall = 2) auf den oberen und unteren Rahmen.
3. Im nächsten Schritt definieren Sie den Abstand der horizontalen Sprosse vom Scheibenrand. Klicken Sie auf die Schaltfläche **Relativ** und anschließend auf den Fangpunkt des oberen Scheibenrandes. Es öffnet sich die Dialogbox **Relativer Abstand**. Unter **Relativ-DX** geben Sie den Abstand von 0 an, da sich der neue Fangpunkt auf der x-Achse nicht verändert. Bestätigen Sie nun die Angaben, werden Sie zur Eingabe von **Relativ-DY** aufgefordert. Hier geben Sie den Abstand von 300 mm an. Da sich die horizontale Sprosse unterhalb des Fangpunktes (am oberen Scheibenrand) befindet, müssen Sie eine -300 eingeben. Dieser Wert ist editierbar und wird bei erneutem Laden des Makros abgefragt. Erfassen Sie bitte einen Text als Erklärung; in unserem Beispiel **Abstand Scheibenrand bis Sprossenmitte**.
4. Im nächsten Schritt setzen Sie die Fangpunkte am rechten und am linken Scheibenrand. Sie beginnen mit dem linken Scheibenrand. Klicken Sie wieder auf **Relativ** und anschließend auf den oberen Fangpunkt des linken Scheibenrandes. Es öffnet sich erneut die Dialogbox **Relativer Abstand**. Der Relativ-DX ist wieder 0. Beim Relativ-DY klicken Sie auf **Wie andere Eingabe** und anschließend auf [OK]. Es öffnet sich die Dialogbox **Auswahl vorhandener Eingaben**. Wählen Sie den Eintrag **Abstand Scheibenrand bis Sprossenmitte**. Bestätigen Sie die Auswahl mit [OK].
5. Setzen Sie nun den noch fehlenden Fangpunkt auf dem rechten Scheibenrand.
6. Anschließend setzen Sie die Sprossen ein. Aktivieren Sie zunächst die Schaltfläche **Sprosse**, klicken Sie dann auf den linken und anschließend auf den rechten Fangpunkt des Scheibenrandes. Fügen Sie nun die noch fehlende senkrechte Sprosse ein.
7. Wählen Sie nun im Menü **Makro** den Eintrag **Speichern**. Es öffnet sich der Dialog **Makro speichern**. Geben Sie den Namen **T-Sprosse** ein.
8. Betätigen Sie die Schaltfläche [OK]. Das Makro ist gespeichert.

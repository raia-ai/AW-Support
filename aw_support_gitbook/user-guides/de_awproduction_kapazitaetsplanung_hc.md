Schichtpläne                                                                          Schichtplan erstellen   KAP 02-001


Ziel der Handlung
Sie erstellen einen Schichtplan.


Voraussetzungen
•   Schicht muss angelegt sein.


Zusatzinfo
Ein Schichtplan kann erst gespeichert werden, wenn ihm eine Schichtregel und eine
Schichtgruppe mit Maschinen zugeordnet wurden!




Workflow
1 Wählen Sie Stammdaten > Kapazitätsplanung > Schichten.
2 Markieren Sie im Bereich Schichteditor die Option Kalender.
3 Klicken Sie auf die Auswahl-Liste [Neu].
4 Wählen Sie Neuer Schichtplan.
    Im Bereich Schichteditor wird ein neuer Schichtplan mit der Bezeichnung Neu an-
    gelegt.
    Alternativ öffnen Sie mit einem Rechtsklick auf den Kalender das Kontext-Menü
    und wählen die Eintrag Neuer Schichtplan.
5 Geben Sie dem neuen Schichtplan im Feld Schichtplan im Feld Name eine Be-
  zeichnung, z. B. Zwei-Schicht normal.
6 Klicken Sie auf [Übernehmen], um die Daten zu speichern.
7 Klicken Sie auf [Erzeugen], damit die Daten des Schichtplans erzeugt werden.


  Mo-Fr für eine Fünf-Tage-Arbeitswoche.

Sie bearbeiten die Lastverteilung.


Voraussetzungen
•   Lastverteilung muss stattgefunden haben.


Zusatzinfo




Workflow                                                                              7 Klicken Sie auf [OK], um die Daten zu speichern und den Dialog Lastverteilung zu
                                                                                        schließen.
1 Wählen Sie Stammdaten > Kapazitätsplanung > Lastverteilung.
                                                                                      
2 Wählen Sie im Feld Physikalische Maschinen eine Maschine, deren Kapazität Sie
  aufteilen möchten, z. B. 910 - CNC Center.
    In der Auswahl-Liste sind nur die Maschinen zur Auswahl aktiviert, die als Eng-
    passbetriebsmittel definiert wurden.
    Die zugehörigen logischen Maschinen werden in der Liste angezeigt.
3 Aktivieren Sie die Checkbox Prüfung aktivieren.
    Die logischen Maschinen können jetzt bearbeitet werden. Wurde noch keine
    Lastverteilung definiert, steht in der rechten Spalte unbegrenzt.
4 Klicken Sie doppelt in die rechte Spalte.
5 Geben Sie den Prozentwert für die Kapazität ein.
6 Wiederholen Sie die Schritte 4 bis 5, um den restlichen logischen Maschinen die
  Kapazitäten einzurichten.

Lastverteilung                                                                            Lastverteilung löschen                                             KAP 11-003


Ziel der Handlung
Sie löschen eine Lastverteilung.


Voraussetzungen
•   Lastverteilung muss stattgefunden haben.


Zusatzinfo




Workflow                                                                                  7 Wiederholen Sie die Schritte 4 bis 6, um die restlichen Kapazitäten auf den Wert
                                                                                            000% zu setzen.
1 Wählen Sie Stammdaten > Kapazitätsplanung > Lastverteilung.
                                                                                          8 Klicken Sie auf [OK], um die Daten zu speichern und den Dialog Lastverteilung zu
2 Wählen Sie im Feld Physikalische Maschinen eine Maschine, deren Kapazität Sie             schließen.
  löschen möchten, z. B. 910 - CNC Center.
                                                                                              Damit haben Sie die Lastverteilung für die Maschine gelöscht
    Die zugehörigen logischen Maschinen werden in der Liste angezeigt.
                                                                                          
3 Aktivieren Sie die Checkbox Prüfung aktivieren.
    Die logischen Maschinen können jetzt bearbeitet werden.
4 Klicken Sie doppelt in die rechte Spalte.
5 Geben Sie 000% ein.
6 Drücken Sie die <Eingabe>-Taste.
    In dem Feld steht jetzt unbegrenzt, damit ist die Lastverteilung für diese logische

Ziel der Handlung
Sie bearbeiten Aufträge nach, deren Einlastung fehlgeschlagen ist.


Voraussetzungen
•   Einlastung ist fehlgeschlagen.


Zusatzinfo




Workflow                                                                                  7 Wiederholen Sie die Schritte 4 bis 6, um die restlichen Kapazitäten auf den Wert
                                                                                            000% zu setzen.
1 Wählen Sie Stammdaten > Nachbearbeitung Einlastung.
                                                                                          8 Klicken Sie auf [OK], um die Daten zu speichern und den Dialog Lastverteilung zu
2 Wählen Sie im Bereich Anzeigen nach, ob die Aufträge nach Auftragsnummern                 schließen.
  oder Nachbearbeitungsursachen angezeigt werden sollen.
                                                                                             Damit haben Sie die Lastverteilung für die Maschine gelöscht
3 Wählen Sie im Bereich Nachbearbeitung von, welche Aufträge angezeigt werden
  sollen.                                                                     
4 Markieren Sie den Auftrag, den Sie bearbeiten wollen.
5 Klicken Sie auf [Bearbeiten].
    Der Dialog Fehlerhafte Aufträge oder der Dialog Änderung bereits verplanter Auf-
    träge wird geöffnet.
6 Drücken Sie die <Eingabe>-Taste.
    In dem Feld steht jetzt unbegrenzt, damit ist die Lastverteilung für diese logische
    Maschine aufgehoben.
